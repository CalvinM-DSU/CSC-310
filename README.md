# A0-BST_Review
This is the integration of a BST that I made in Tom's Data Structures class last semester. It reads a file as a command line argument which gives the instructions.

Instructions:
1 x   Insert
-1 x  Delete
2 x   Search (returns y/n)
3     Preorder
4     Inorder
5     Postorder
7     Number of Nodes
8     Height of tree
9     Compares Left and Right height
0     Quits the program
